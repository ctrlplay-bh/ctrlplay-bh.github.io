Dentre os tipos de display possíveis de serem utilizados podemos separar os mais importante em duas categorias:

Display-Outside
É responsável por descrever o comportamento do layout entre dois elementos adjacentes.
Display-Inside
É responsável por descrever o comportamento do layout entre um elemento pai e seus elementos filhos.

    <Display-Outside>


  • Block
  • Inline

Flow Layout

Os elementos com display block ou inline participam do layout padrão do CSS. Veremos como isso funciona para cada tipo de display.

Display: Block

Block

Um elemento com display block gera uma quebra de linha antes e depois do elemento


 <div style="background-color:#08B">
   Algum texto
 </div>
 <div style="background-color:#F04">
   Outro texto
 </div>
Algum texto
Outro texto

Block

Por padrão ele ocupa todo o width disponível para ele.

 </div>

Block

Quando a margem superior do bloco 2 fica maior, percebe-se uma diferença

Isso é o margin-collapse.

O margin-collapse somente ocorre entre margens verticais!

Você deve ter reparado no espaço sem cor entre a div com fundo azul e fundo rosa, certo?

Isso também é um caso de margin-collapse, porém ela ocorre entre a margem dos elementos pai e filho

Uma das várias maneiras de impedir isso, é colocando um padding de um 1px na div externa. Isso irá separar as margens de elementos pai e filho. 



  <div style="background-color:#08B; width: 25%">
    <div style="background-color:#3A3; font-size: 0.7em; margin: 15px;">
      Bloco interno 1
    </div>
    <div style="background:#771; font-size:0.7em; margin:30px">
      Bloco interno 2

    Outro texto

Esse é o modo padrão de funcionamento do CSS

Considerando o início da internet, ele era um padrão normal de se pensar

As páginas eram simples, contendo apenas imagens, links e texto

Isso foi mudando e era necessário páginas mais estilizadas

Era preciso layouts mais modernos, visivelmente agradáveis

Surgiram várias técnicas para se fazer isso

Usando tables, usando floats e vários outros hacks

Até a chegada do CSS3

Flexbox e Grid

Os displays flexbox e grid são as versões mais novas e eficientes de criar o layout do site

É muito mais simples criar páginas estilizadas através deles. Além disso, as funções específicas para o uso dos floats, por exemplo, faz muito mais sentido
