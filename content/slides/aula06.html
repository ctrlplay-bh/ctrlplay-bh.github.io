Além do posicionamento dos elementos ser regido pelo tipo do display utilizado, outra propriedade importante é o position.

Ela especifica a posição de um elemento individual na tela. Depois de especificar o tipo do position podemos passar valores para: top, bottom, left e right

Position: Static

Esse é o position padrão dos elementos na página.

Este tipo de posicionamento não é afetado pelos valores de top, bottom, left e right

Static

O posicionamento static segue o flow

Static

Ele não é afetado por top, bottom, left, right

Relative

Um elemento com esse atributo é posicionado relativamente a sua posição normal

Outro exemplo

Absolute

Um elemento com esse atributo é posicionado relativamente ao seu ancestral

Porém o elemento ancestral deve ter um posicionamento(não pode ser static)

Nesse caso o posicionamento será relativo ao body

Absolute

Veja a diferença

Um elemento com posicionamento absolute é removido do flow e por isso o elemento pai fica vazio

Fixed

Um elemento com position: fixed é posicionado em relação ao viewport e será mantido sempre naquela posição.
